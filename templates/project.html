О проекте

Проект создан на Python и современном веб-стеке

Сайт реализован на Python с использованием Flask, HTML, CSS и JavaScript. Дизайн вдохновлён современными маркетплейсами, такими как Wildberries, Ozon и Amazon, но при этом имеет собственный стиль и яркий внешний вид.

Проект был создан как многостраничный сервис с каталогом, поиском, карточками товаров, корзиной, личными настройками и разделом для информации о разработчике. Все основные элементы были собраны вручную, а данные хранятся в JSON для удобного расширения.
